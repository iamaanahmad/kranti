# Contributing to Kranti

Thanks for your interest in contributing.

## Ways to help

- Improve civic-facing copy and accessibility.
- Fix bugs and improve the user experience.
- Add tests, docs, and reliability improvements.
- Help strengthen moderation, privacy, and safety flows.

## Local setup

```bash
npm install
npm run dev
```

## Before you submit

- Run `npm run build`.
- Run `npm run lint` if your change touches shared code.
- Keep changes focused and avoid unrelated formatting noise.

## Pull requests

- Explain what changed and why.
- Link any related issue or discussion.
- Include screenshots for visible UI changes.

## Code style

- Prefer small, focused components.
- Match the existing Next.js and Tailwind style in the repository.
- Keep public-facing copy simple, factual, and user-centered.
